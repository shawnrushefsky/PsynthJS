Class: Graph

Psynth. Graph

new Graph(params)

This constructor cannot be directly accessed, but instead should be accessed through the Psynth.createGraph or Psynth.loadGraph methods.
Parameters:
Name Type Description
params object
Properties
Name Type Description
name string The name of this Graph.
filename string The filename of this Graph. Must be unique to your Psynth server. This is assigned by the server.
username string A Psynth username authorized to access the graph.
key string A key which matches the username.
url string The base URL for your Psynth server. e.g. https://psynth.psymphonic.com/
Source:
  • psynth.js, line 47
Example
var g = new Graph({name: 'My Graph', filename: 'b8e1241a-c90c-46ca-a55e-6cbb9145ab19.gt',
                          username: 'myusername', key: 'mykey', url: 'https://psynth.psymphonic.com'}

queue(q, handler)

All queries should be queued through this function to ensure that they complete in order. Not doing this can result in links trying to attach to non-existent nodes.
